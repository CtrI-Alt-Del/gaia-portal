## Comece entendendo o problema

Eventos climáticos extremos são cada vez mais frequentes. Muitas escolas ficam sem acesso a dados locais sobre chuva, vento ou variações de temperatura e, por isso, têm dificuldade em discutir risco, prevenção e impacto ambiental de forma prática.

O Gaia nasce para aproximar escolas da realidade da própria região, oferecendo:

- Coleta de dados ambientais com baixo custo
- Visualização amigável para estudantes
- Contexto para projetos de investigação científica em sala de aula

## Do sensor ao portal

O fluxo básico do ecossistema Gaia é:

1. Sensores conectados a um Arduino coletam variáveis climáticas.
2. O datalogger envia os dados para um broker MQTT, como o HiveMQ.
3. Serviços em NodeJS consomem as mensagens, tratam os dados e salvam em bancos como MongoDB e PostgreSQL.
4. O Gaia Panel exibe dashboards, relatórios e ferramentas de análise para estudantes e educadores.

## Construído para experimentação e desenvolvimento

O Gaia foi pensado para ser extensível por estudantes e equipes técnicas.

Isso traz vantagens como:

- Controle total sobre o código do datalogger, backend e portal
- Possibilidade de adicionar novos sensores e camadas de visualização
- Integração com outras ferramentas de análise de dados
- Liberdade para adaptar o projeto à realidade de cada escola ou município
